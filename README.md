# Recalibrate 17 Landing Page

This project is a fresh React + Vite setup based on your provided style language.

## Run locally

```bash
npm install
npm run dev
```

## Replace placeholders

- `src/App.jsx` contains three product placeholder cards in the `products` array.
- Replace the image placeholder blocks with your final product images.
- Replace `R17` / `RECALIBRATE 17` with final logo and brand assets.

## Tech stack

- React 18
- Vite
- Lucide React (outline icons)
